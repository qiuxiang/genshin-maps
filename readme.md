## web 版

由于 web 的一些限制，必然做不到和原生 app 一样的体验，但在做一些针对优化后流畅度也足以保证。

flutter 在 web 可以使用 canvaskit 进行渲染，大体上还原原生的体验，当然，由于 web 及 js 性能限制，
web 版必然比原生更容易出现掉帧，但也远远优于 dom 实现。
