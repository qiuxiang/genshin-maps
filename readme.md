**目前在对地图渲染和 UI 做一些小范围重构，预计年后开源维护，欢迎 issues 讨论，不过精力有限功能会做得比较慢。**

使用 flutter 实现，极致流畅的原神地图 app，可以离线使用，支持 android、web、linux、windows、macos。为了提供最佳的用户体验，从手势识别到资源加载、地图绘制都是独立实现的。
